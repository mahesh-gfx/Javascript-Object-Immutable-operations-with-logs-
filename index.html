Javascript Object Immutable Operations with Log in Local Storage

Instruction: Open Inspector and go to Console

To Create Object:
createObject(object);
Example: createObject({name: "Mahesh", age: 22});

To Update Object:
updateObject(object, key, value);
Example: updateObject({object, "name", "Mahesh");

To Delete Key from Object:
deleteKeyFromObject(object, key);
Example: deleteKeyFromObject({object, "name");

To Freeze Object:
freezeObject(object);
Example: freezeObject(object);

To Seal Object:
sealObject(object);
Example: sealObject(object);

To Prevent Object Extensions:
preventExtensions(object);
Example: preventExtensions(object);
